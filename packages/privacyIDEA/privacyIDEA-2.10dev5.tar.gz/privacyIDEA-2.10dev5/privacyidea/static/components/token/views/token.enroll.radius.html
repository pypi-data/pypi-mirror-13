The RADIUS token forwards the authentication request to another RADIUS server. You can choose if the PIN should be stripped and checked locally.

Token data

To ease the enrolling of RADIUS tokens you can set default values in the RADIUS Token Config.

If the PIN is split and verified locally, you need to define, how long the password is, that is sent to the RADIUS server.

The RADIUS server may include the port number.

This is the username that is sent to the RADIUS server. It can be different from the owner of this RADIUS token.
