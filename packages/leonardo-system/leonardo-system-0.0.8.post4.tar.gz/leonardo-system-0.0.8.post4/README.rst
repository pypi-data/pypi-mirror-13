
==========================
Leonardo leonardo-system
==========================

System Module to Leonardo CMS

Features
--------

* install/uninstall modules from pypi in one click
* check/update versions

.. contents::
    :local:

Requirements
------------

* pip 1.5 +

Installation
------------

.. code-block:: bash

    pip install --upgrade pip
    pip install leonardo-system
