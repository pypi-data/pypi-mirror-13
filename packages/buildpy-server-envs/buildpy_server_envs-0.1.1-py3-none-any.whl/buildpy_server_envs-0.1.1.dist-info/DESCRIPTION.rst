`Buildpy-server-envs`_ is a plugin for `buildpy-server`_.
A simple plugin to use with Buildpy-Server.

Installation
------------

You can install "buildpy-server-testcommands"
via `pip`_ from `PyPI`_::

    $ pip install buildpy-server-envs
