tldr-python-client |Build Status|
=================================

A ``Python`` command line client for
`tldr <https://github.com/tldr-pages/tldr>`__.

.. figure:: http://raw.github.com/tldr-pages/tldr/master/screenshot.png
   :alt: tldr screenshot

   tldr screenshot

Installation
------------

::

    $ pip install tldr

Usage
-----

-  ``tldr <command>``
