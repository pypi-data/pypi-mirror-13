`Buildpy-server-testcommands`_ is a plugin for `buildpy-server`_.
It provides running test commands. 

Installation
------------

You can install "buildpy-server-testcommands"
via `pip`_ from `PyPI`_::

    $ pip install buildpy-server-testcommands
