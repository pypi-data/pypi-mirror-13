

Run the Tests
-------------

Install tox and run it::

    pip install tox
    tox

Limit the tests to a specific python version::

    tox -e py27

Conventions
-----------

Onegov Onboarding follows PEP8 as close as possible. To test for it run::

    tox -e pep8

Onegov Onboarding uses `Semantic Versioning <http://semver.org/>`_
